== Description ==
Very simple plugin to hide 'non essential' wp-admin functionality.
This plugin is intended for select client projects where the client is an editor.


Features include:
1 Hides the dashboard and directs users to the edit pages screen 
2 Hides non essential menu items & separators 
3 Hides columns on the edit page or post overview screens 
4 Hides the footer as well various other minor css changes 
5 Hides the WordPress admin bar (only front end) 


Options to:
1 Hide Posts from the WordPress menu
2 Hide screen options tab, help tab and the post filtering bar  


Works well with:
1 WordPress SEO: Hides SEO columns from the edit page / edit posts screens 
2 Gravity Forms: Grants editor (client) access to manage Gravity Forms 


== Installation ==

Upload Minimal Admin to /wp-content/plugins/

Activate the plugin via the plugins menu
